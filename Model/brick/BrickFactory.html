Class BrickFactory

Package: main.java.model.brick

Author: Emily, Updated by Anvitha Ugrani

Description

This class implements the Factory Design Pattern to create and customize brick objects for the Brick Breaker game. It allows flexible creation of different brick types like Clay, Steel, Cement, and the new GlassBrick.

Method Details

makeBrick(Point point, Dimension size, BrickType type)

Description: Creates a brick object based on the provided type.


switch(type) {
    case CLAY:
        return new BrickController(new ClayBrick(point, size));
    case STEEL:
        return new BrickController(new SteelBrick(point, size));
    case CEMENT:
        return new BrickController(new CementBrick(point, size));
    case GLASS:
        return new BrickController(new GlassBrick(point, size));
    default:
        throw new IllegalArgumentException("Unknown brick type!");
}

makeRandomBrick(Point point, Dimension size)

Description: Randomly selects a brick type and returns a new brick instance.


BrickType[] types = BrickType.values();
BrickType randomType = types[new Random().nextInt(types.length)];
return makeBrick(point, size, randomType);

makeCustomBrick(Point point, Dimension size, BrickType type, Color color, int durability)

Description: Allows customization of brick color and durability for more flexible gameplay design.


BrickController brick = makeBrick(point, size, type);
brick.setColor(color);
brick.setDurability(durability);
return brick;

New Feature: GlassBrick

GlassBrick is a new lightweight brick type with low durability but high visual transparency. It breaks easily, rewarding players with bonus points.

Summary

  • Supports 4 brick types: Clay, Steel, Cement, and Glass.
